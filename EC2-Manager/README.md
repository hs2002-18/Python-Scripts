# EC2 Manager 

A simple command-line EC2 management tool built with Python and Boto3.

This script allows users to:
- List all EC2 instances and their current state
- Stop a specific EC2 instance
- Create a new EC2 instance with a predefined AMI and instance type

The tool uses a menu-driven interface to perform basic EC2 operations in the `ap-south-1` region.
